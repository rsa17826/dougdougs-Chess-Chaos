# Chess Game Server

A minimal Python HTTP server for the chess game backend.

## Running the Server

```bash
python3 chess_server.py
```

Server will run on `http://127.0.0.1:8000`

## Configuration

Edit these in `chess_server.py`:

```python
GAME_PASSWORD = "password123"  # Change this to your desired password
PORT = 8000                     # Change this if port 8000 is in use
```
